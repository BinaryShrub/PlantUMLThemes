# Plant UML Themes

## How To Use
Add the following `!include` statement following `@startuml`

```
!include https://raw.githubusercontent.com/BinaryShrub/PlantUMLThemes/main/BlueGray.puml
```

Example:
```
@startuml
!include https://raw.githubusercontent.com/BinaryShrub/PlantUMLThemes/main/BlueGray.puml

.
.
.

@enduml

```
